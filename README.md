Bulls and Cows Game

A fully functional console-based implementation of the classic code-breaking game Bulls and Cows. Players attempt to guess a 4-digit secret code with feedback provided in the form of Bulls (correct digit in correct position) and Cows (correct digit in wrong position).
Features

Support for multiple player types:
Interactive Human Player (with input validation)
Easy AI (random guessing)
Medium AI (remembers previous guesses)
Hard AI (strategic guessing using elimination)
File-based Player (reads secret code and guesses from a text file)

Complete game logic including 7-guess limit, win/loss/draw conditions
Input validation to ensure only valid 4-digit codes with no repeating digits are accepted
Game results saved to a text file upon completion
Clean object-oriented design with inheritance and multiple classes
UML class diagram included

Technologies

Java 17

Object-Oriented Programming (OOP)

File I/O

Console-based user interface
